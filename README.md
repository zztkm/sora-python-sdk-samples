## Requirements

- [PDM](https://pdm-project.org/latest/) を使います。
    - インストール例: `rye install pdm`

## Commands

### Install dependencies

pdm で使うインタプリターを選択します。
参考: https://pdm-project.org/latest/usage/project/#choose-a-python-interpreter
```bash
pdm use
```

依存関係をインストールします。

```bash
pdm install
```
